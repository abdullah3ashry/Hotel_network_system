# Hotel Network System

This project contains a network simulation designed for a hotel environment using Cisco Packet Tracer.

## Overview
The `hotel_system.pkt` file provides a structured network topology tailored to meet the connectivity, security, and management requirements of a modern hotel. The simulation includes configurations for various departments, guest services, and administrative operations.

## Features
- **Departmental Segmentation:** Uses VLANs to separate traffic between Guest Wi-Fi, Administrative offices, Security/Surveillance, and Front Desk operations.
- **Network Services:** Includes configurations for essential services such as DHCP for automatic IP assignment, DNS for internal mapping, and potentially Web/Email servers for hotel management.
- **Security:** Implements basic access control lists (ACLs) to ensure that guest traffic remains isolated from sensitive internal hotel data.
- **Connectivity:** Demonstrates a blend of wired (Ethernet) and wireless (WLAN) connectivity to support both back-office systems and guest mobility.

## Prerequisites
To view and interact with this project, you will need:
- **Cisco Packet Tracer** installed on your machine.

## How to Use
1. Download the `hotel_system.pkt` file.
2. Open Cisco Packet Tracer.
3. Go to `File > Open` and select the `hotel_system.pkt` file.
4. Explore the topology in the logical workspace.
5. Use the "Simulation" mode to observe packet flow across different VLANs and network devices.
